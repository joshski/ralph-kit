# ralph-kit

An opinionated [Ralph with beads](https://www.chrismdp.com/your-agent-orchestrator-is-too-clever/) starter project, with:

* [bun](https://bun.com/)
* [biome](https://biomejs.dev/)
* [typescript](https://www.typescriptlang.org/)

To get started:

```
bun install
bd init
```
